# Podología App
Proyecto listo para desplegar en Vercel (frontend estático + funciones serverless).

Instrucciones rápidas:
1. Configura variables de entorno en Vercel:
   - DB_SERVER
   - DB_USER
   - DB_PASSWORD
   - DB_DATABASE
   - JWT_SECRET
2. Sube este repo a GitHub y conéctalo en Vercel.
3. Ejecuta `vercel dev` para probar localmente.

Archivo SQL incluido: `database.sql`. Este es el script que crea las tablas.
